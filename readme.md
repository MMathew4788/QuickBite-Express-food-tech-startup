# QuickBite Express Crisis Recovery Dashboard

## Overview

This repository contains a Power BI dashboard project designed to analyze and visualize the impact of a business crisis on QuickBite Express, a food delivery startup. The dashboard aims to provide actionable insights for business recovery and strategic decision-making.

## Getting Started

- Clone this repository to your local machine.
- Open the Power BI (.pbix) file included in the project.
- Review the data model and dashboard structure.
